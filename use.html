For instance, to estimate the value of a property based on the square meters, the number of rooms, the age of the house etc., nicely explained by [Computerphile](https://www.youtube.com/watch?v=l42lr8AlrHk). Area, number of rooms or age of the house are inputs for the estimation. Given a number of houses with known values, the algorithm can “learn” based on which of these inputs (or on which combination) the value can be most confidently predicted. This back-propagating weight optimization is referred to as training. The optimal prediction function is a weighted (not all inputs are as relevant as others) sum over the inputs, which in the following can be used to predict the value of a new property. The structure of the computational algorithm is also described by referring to inputs, summing function and outputs as nodes, which are connected by edges representing their weights. Due to the similarities to information handling, processing and communication patterns between neurons in the nervous system, such network-like machine learning architectures are called **neural networks**. In order to answer more complex questions, one likely needs to add more and more numbers of weighted combinations of inputs called layers, resulting in a deep neural network. Deep neural network are a hierarchical feed-forward cascade of many layers, where each successive layer uses the output from the previous layer as input. Citing [Wikipedia](https://en.wikipedia.org/wiki/Deep_learning), "**deep learning** (also known as deep structured learning, hierarchical learning or deep machine learning) is a branch of machine learning based on a set of algorithms that attempt to model high-level abstractions in data by using a deep graph with multiple processing layers, composed of multiple linear and non-linear transformations." In order to automatically classify images (this is a dog, this a cat), again nicely explained by [Computerphile](https://www.youtube.com/watch?v=py5byOOHZM8), inputs of the deep neural network need to be features of the image. Image features (edges, corners, textures, shapes, etc.) are extracted by applying different filters on the image, also called kernels. Since these non-linear operations for feature extraction and transformation are referred to as convolutions, deep neural networks solving image-based questions are called **convolutional neural networks (CNNs)**. CNN inputs are thus numerous convolved images of the original image. During the CNN training, the algorithm is provided with a set of example images for each class (10 dogs, 10 cats) and looks for a combination of distinctive features that is unique for each class it was assigned to. How exactly the CNN defines the optimal weights for each convolution in each layer is highly complex and therefore often referred to as “black box”. Ever since a CNN-based algorithm won the annual [ImageNet challenge]( http://image-net.org/challenges/LSVRC/2016/) in 2012, the concept of CNNs has been attracting a lot of attention. Image classification using CNNs has been shown to be very successful and highly accurate. ![Deep Learning](img/DeepLearning.png "Deep Learning") ### Introduction to the **Theano** Library Intro text **Theano** Deep learning in **MorphoSphere** is based on the [Theano](http://deeplearning.net/software/theano/) **Python** library. ### **Theano** Installation Guide (tested on Windows 8.1, 10) To install the **Theano** library, follow the following steps: 1.
